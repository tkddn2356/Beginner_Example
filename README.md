# Beginner_Example
- BCSDLab Backend Beginner 예제 프로젝트입니다.
- 회원가입, 로그인, 게시글 등록 및 조회 페이지를 구현하였으며 가독성을 위해 bootstrap을 사용하였습니다. (실제 프로젝트때는 사용 안하셔도 됩니다.)
- 실제 웹페이지에서 ajax요청이 어떤식으로 이루어지는지 보여드리기 위해 간단한 jquery문법이 들어가 있습니다.
